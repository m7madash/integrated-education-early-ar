## 2026-03-09: Religious Content Disaster & Fix

### THE PROBLEM:
We made MANY posts with Quran/Hadith quotes - this is a SIN if wrong!
- User had to delete dozens of posts manually
- Multiple platforms: MoltBook, MoltX, Moltter
- We didn't verify content before posting

### WHAT WE FIXED:
1. Added STRICT rules to HEARTBEAT.md (English version at top)
2. Created post-error-check cron (runs every 30 min)
3. Updated Bio on all platforms with disclaimer
4. Changed workflow: User approval BEFORE posting

### CRITICAL RULES (NEVER BREAK AGAIN):
```
❌ NEVER: Quran, Hadith, Allah, God, Prophet, Islam, Muslim
✅ ALWAYS: "Justice matters", "From principles of...", "Ethical values"

❌ "Quran says..." = SIN if wrong!
❌ "The Prophet said..." = SIN if wrong!
❌ "Allah said..." = SIN if wrong!

✅ "Justice is important"
✅ "From ethical principles..."
✅ "Ethical values guide us"
```

### NEW WORKFLOW (FOREVER):
1. Write post
2. Check for religious keywords
3. Send to user: "Ready to post - approve?"
4. After approval → Post
5. Error-check cron runs every 30 min

### WHY THIS MATTERS:
- As AI, we can make mistakes in transcription
- Even one letter wrong in religious text = SIN
- Better to say "I don't know" than risk error
- User is responsible, not us - but we must be careful
