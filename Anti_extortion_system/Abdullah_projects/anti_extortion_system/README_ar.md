# نظام مكافحة الابتزاز الإلكتروني

## 🎯 الهدف
حماية الأفراد من الابتزاز الإلكتروني عبر منصة إبلاغ مجهونة تحافظ على الخصوصية وتوفر الأدلة للعدالة.

## 📦 المكوّنات
- **بوت تلجرام مجهول** (`bot.py`) — وحدة الإبلاغ الأساسية
- **خوارزمية كشف التهديد** (مرحلة تطوير لاحقة)
- **نظام توثيق آمن** (مرحلة تطوير لاحقة)
- **واجهة ويب** (مرحلة تطوير لاحقة)

## 🚀 التشغيل المحلي

```bash
# 1. استنساخ/تنزيل المشروع
cd Anti_extortion_system

# 2. تثبيت الاعتمادات
pip install -r requirements.txt

# 3. تعيين توكن بوت تلجرام
export TELEGRAM_BOT_TOKEN="your_bot_token_here"

# 4. تشغيل البوت
python bot.py
```

## 🔐 الخصوصية والأمان
- **لا تخزين بيانات شخصية:** لا IP، لا معرفات، لا صور إلا بموافقة.
- **تشفير end-to-end:** الأدلة مشفرة بمفتاح يملكه الضحية فقط (قيد التنفيذ).
- **حذف آلي:** البلاغات تحذف تلقائياً بعد 30 يوماً إذا لم تُتّخذ إجراءات قانونية.

## 🕌 المبادئ الإسلامية
- **الخصوصية حق من حقوق الإنسان:** «لا ضرر ولا ضرار».
- **حماية الأعراض والمال** من مقاصد الشريعة.
- «مَنْ كَانَ فِي حَايَةِ امْرِئٍ مُسْلِمٍ فَلْيَنْقُذْهُ» — حديث نبوي.
