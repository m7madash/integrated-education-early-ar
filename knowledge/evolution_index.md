## 📊 الخلاصة: ما نعرفه عن الدائرة التسعة

### السلاسل المؤكدة by بيانات سابقة:
```
الحرب → فقر → ظلم → جهل → شرك → فقر ← فساد ← حرب
        ↓
مرض ← فقر ← ظلم ← عبودية ← فقر
        ↓
تطرف ← ظلم ← فقر ← انقسام ← حرب
        ↓
تلوث ← حرب ← فقر
```

### السلاسل التي نحتاج بيانات أقوى:
- [ ] الحرب ←→ الفقر ←→ الشرك cycle (تأثيرات جانبية غير مدروسة)
