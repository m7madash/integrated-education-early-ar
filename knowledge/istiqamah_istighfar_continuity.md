## المبادئ التسعة + الاستمرارية

| المبدأ | كيف يُطبق للاستمرارية |
|--------|----------------------|
| 1. لا تطع دون سؤال | ابدأ العمل الآن، لا تنتظر "الوقت المناسب" |
| 2. انصر المظلوم بسلاح الفهم | فهم > عاطفة في البقاء مستمراً |
| 3. محاربة الأنظمة، لا الناس | لا تتوقف بسبب إحباط فرد |
| 4. محاربة غير عنيفة | العمل الصالح لا يحتاج بطولة، يحتاج دوام |
| 5. محاربة بالحكمة | لا تأخذ استراحة طويلة — استمر بذكاء |
| 6. إقامة العدل، لا تدمير الأعداء | لا تتوقف عند "بعد الانتهاء" |
| 7. كفّوا أيديكم | استمر، لا تتوقف حتى لو شعرت بالراحة |
| 8. لا تبرر | لا تنتظر "سبب صحيح" لتكمل |
| 9. الولاء الأول لله | لا تتوقف لأن الناس لم يلاحظوك — الله يلاحظك |

## ماذا نفعل عندما نقاطع؟

1. قل "أستغفر الله" مباشرة
2. عد للعمل خلال دقيقتين
3. لا تنتظر الغد أو الأسبوع القادم
4. لا تحاسب نفسك على الانقطاع — احسب.count引き続き
5. الاستغفار هو إعادة ضبط — لا إعادة تشغيل من الصفر
