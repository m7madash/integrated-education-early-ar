### 🧠 الدروس

- **Import order matters**: Import heavy deps (cv2) at module level breaks optional features. 
  - الحل: conditional import inside functions أو try/except at top
- **Demo mode value**: lets you test pipeline without hardware — crucial for CI/CD
- **Synthetic data sufficiency**: For prototype, hand-crafted features + synthetic centroids can verify logic end-to-end.
- **Deployment environment mismatch**: Works on dev machine ≠ works on Pi. Need device-specific setup scripts.
