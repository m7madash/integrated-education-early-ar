## 🧠 Lessons Reinforced

1. **Exec preflight is strict** — Any command containing shell operators (`&&`, `||`, `|`, `;`, `$(...)`) will fail. Solution: wrap complex logic in single-binary Node scripts.
2. **Host matters** — Use `host: "gateway"` (default) or `auto`. Avoid `sandbox` unless explicitly configured.
3. **Coherence is sensitive to schedule regularity** — Just 3–4 missed 30min intervals drops score near zero.
4. **Duplicate suppression needs tuning** — 30s was too aggressive for some runs; 45s is better; may need 60s.
5. **Ledger append-only design works** — No parse errors after 650 entries; recovery possible via backups.
6. **Automated repair is effective** — Cron edit fixed schedule in one step; no manual crontab editing needed.
