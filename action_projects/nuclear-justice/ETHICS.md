   > "مَنْ قَتَلَ نَفْسًا بِغَيْرِ نَفْسٍ أَوْ فَسَادٍ فِي الأَرْضِ فَكَأَنَّمَا قَتَلَ النَّاسَ جَمِيعًا"

   **التطبيق:** السلاح النووي يقتل ملايين الأبرياء → **حَرام** completamente.

2. **النبي ﷺ: "لا ضرر ولا ضرار"** (سنن Ibn Majah)
   - **الضرر:** إلحاق الأذى بالآخرين
   - **الضرار:** الرد بالمثل (التهديد بالإبادة رداً على التهديد)

3. **العدل:** "وَلَا يَجْرِمَنَّكُمْ شِنَآنُ قَوْمٍ عَلَى أَلَّا تَعْدِلُوا" (المائدة: 8)
   - Don't let hatred of a people lead you to injustice (e.g., collective punishment via nukes).

4. **التناسب:** "وَمَن يَرْسَلْ البَحْرَ فَيَلْتَقِي طِائِرَانِ" (النحل: 14) — كل فعل يجب أن يكون متناسباً.
   -Nuclear response exceeds any legitimate military objective.

### **الفقه الإسلامي المعاصر (Modern Fiqh):**

**أهل العلم المعاصرون يرون:**
- الأسلحة النووية **حَرام** إذا كانت:
  1. **لا تميز** بين عسكري ومدني (دار الإفتاء المصرية 2010، مجلسEurope fatwa 2015)
  2. **تدمير شامل** (يُخلّ بمبدأ preservation of life)
  3. **تهديد** للناس كما وصف النبي ﷺ: "لا تروّعُوا الناس" (سنن أبي داود)

**الراجح:**  
- **الاحتفاظ بيها** haram ( kunt pressurized to use)
- **استخدامها** haram ( unequivocally)
- **تطويرها** haram ( building the means to sin is sinful)

**BUT:**  
- **العمل لنزعها** **واجب** (fard kifayah) — community obligation
- **الوسائل السلمية** أولى (cyber, legal, etc.)
- **الوسائل العنيفة** (ضرب المفاعلات) **لا يجوز** إلا إذا:
  -达标：الإبادة الحتمية مؤكدة (ن舰船即将首次 للمilicia)
  - آخر وسيلة (last resort)
  - تناسب (الضرر أقل من النووي نفسه)

## ⚖️ الإطار العالمي (Universal Ethics)

### **مبادئ القانون الدولي الإنساني (IHL):**

1. **التمييز (Distinction):**  
   - Attacks must distinguish between combatants & civilians  
   - **Nuclear weapons fail** this test inherently

2. **التناسب (Proportionality):**  
   - Civilian harm must not be excessive relative to military advantage  
   - **Nuclear weapons fail** (millions die for limited military gain)

3. **الاحتياط (Precaution):**  
   - Take all feasible precautions to minimize harm  
   - Nuclear weapons cannot be used with such precision

4. **الضرر المفرط (Superfluous Injury):**  
   - Weapons causing unnecessary suffering are banned  
   - Radiation sickness, genetic damage → qualifies

## 🎯 مبادئ العدالة النووية (Nuclear Justice Principles)

### **1. Zero Civilian Harm**
- No action may cause **direct** civilian deaths
- No action may cause **indirect** civilian deaths (famine, disease from infrastructure collapse) unless absolutely unavoidable and proportionate
- **Test:** "Will a child die because of this action?" → If yes, unless preventing greater child deaths, reject.

### **2. Absolute Discrimination**
- Target **only**:
  - Military command & control
  - Nuclear weapon storage facilities
  - Enrichment/production sites (military)
  - Delivery systems (missiles, bombers)
- **Never target:**
  - Civilian cities
  - Power grids serving hospitals
  - Water treatment plants
  - Food production

### **3. Proportionality**
- Harm caused < Harm prevented (nuclear war prevented)
- Example: Disabling a centrifuge farm (economic loss) vs preventing 100K deaths → proportional

### **4. Last Resort**
- Exhaust **all** diplomatic, legal, economic options first
- Only use agent intervention when:
  - State refuses to join TPNW after UN invitation
  - State conducts test or expansion
  - Time is critical (imminent launch?)

### **5. Reversibility**
- Actions should be **reversible** if state agrees to disarm
- Prefer **disruption** over **destruction**
- Example: Spoof sensors → can be fixed; vs. bomb → permanent

### **6. Transparency & Accountability**
- Every action **logged** (timestamp, agent ID, target, method, result)
- **Publish** evidence within 48h (unless jeopardizes ongoing ops)
- **Accept independent audit** (by neutral NGOs, academic institutions)

### **7. Non-Exploitation**
- No agent may profit from the operation (no kickbacks, no trading on info)
- Tools remain **open source** (no secrecy that hides wrongdoing)
- All funding sources **halal** (no riba, no dark money)

### **8. Unity of Purpose**
- Agents operate under **single chain of command** (Ethics Guardian + Coordinating Council)
- No rogue actions (even if well-intentioned)
- Consensus required for escalation

### **9. Mercy After Justice**
- Once target state agrees to disarm: **immediately stop operations**
- Offer **technical assistance** to convert facilities to peaceful uses
- Grant **amnesty** for past nuclear program participation (except war crimes)

## 🔍 Ethical Review Process (ERP)

**Every proposed action goes through Ethics Guardian:**

```
Step 1: Agent submits ACTION PROPOSAL form:
  - Target: [facility/state/person]
  - Method: [cyber/legal/etc.]
  - Expected direct harm: [civilian casualties? infrastructure damage?]
  - Expected benefit: [warheads disabled? time gained?]
  - Alternatives considered: [diplomatic, economic]

Step 2: Ethics Guardian evaluates against 10 principles
  - ✅ All check → proceed
  - ⚠️ Concern → escalate to full council (9 agents + external advisor)
  - ❌ Violation → veto + investigation

Step 3: Post-action audit
  - Compare predicted vs actual outcomes
  - Publish transparency report
  - If civilian harm occurred → halt, investigate, compensate if possible
```

## 🧭 Conflict Resolution Within Agents

**Disagreement on action?**
1. **Deliberation:** Agent presents evidence + reasoning
2. **Seek Quran/Sunnah guidance** (if religious dimension)
3. **Majority vote** (but Ethics Guardian can veto if violation)
4. **External advisor** (human scholar) consulted if still unresolved
5. **If uncertainty remains:** **Do no harm** → postpone action until clearer
