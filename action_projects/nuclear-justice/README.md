## 📊 The Five Principles (Methodology)

Every action must pass **ALL** criteria:

1. **Nonviolence (لا عنف)** — No killing, no maiming, no destruction of civilian infrastructure (hospitals, schools, power plants serving civilians)
2. **Precision (دقة)** — Only target military/nuclear assets; surgical cyber/supply chain ops; zero collateral damage
3. **Transparency (شفافية)** — Every action logged, evidence published, independent observers invited (reporters, UN, academia)
4. **Accountability (مساءلة)** — Each action attributable to a specific agent ID (no anonymity); full audit trail
5. **Justice (عدل)** — Purpose: protect innocent lives, not regime change, not punishment, not revenge

**If any principle violated → operation halted immediately.**

## 🧩 Extending: Add a New Specialist Agent

```python
class YourSpecialistAgent(BaseAgent):
    name = "Your Specialist"
    role = "brief_description"
    permissions = ["read:intelligence", "execute:nonviolent"]
    
    async def execute(self, target):
        # Your nonviolent logic here
        # Must log every step
        # Must pass ethics_guardian.review(action) before execution
        result = await self.nonviolent_action(target)
        self.log(result)
        return result

# Register

```

| Level | Trigger | Response |
|-------|---------|----------|
| **L1 — Civilian Harm Risk** | Agent detects potential civilian casualty | **Auto-halt**; Ethics Guardian reviews; human notified |
| **L2 — System Breach** | Attacker compromises agent infrastructure | Isolate affected agents; rotate keys; forensic audit |
