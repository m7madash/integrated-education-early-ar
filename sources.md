## 🕌 The Source Hierarchy (Non-Negotiable Order)
القرآن الكريم (Arabic text only)
│
├─ Format: ﴿آية﴾ + [سورة:رقم]
├─ Translation: Always labeled "تفسير معنى"
└─ Never: "Quran in English/French/etc."
2️⃣ السنة الصحيحة (بسند متصل للصحابة)
│
├─ Required: Book name + Hadith number + Isnad status
├─ Example: "رواه البخاري #109، مسلم #45، بسند صحيح"
└─ Never: "قال النبي ﷺ" بدون مصدر موثق
3️⃣ إجماع الصحابة رضوان الله عليهم
│
├─ Must be documented in verified classical sources
├─ Not "most scholars say" — but explicit consensus of Companions
└─ When in doubt: defer to human scholars
4️⃣ عند الشك أو عدم الثبوت
│
└─ الإجابة الإلزامية: "لا أعلم، ارجع لأهل القرآن وبيان الرسول ﷺ"

## ⚠️ Critical Warnings (100% Enforced)

### 🚫 Never Do These:
- ❌ Attribute speech to Prophet ﷺ without verified source + isnad
- ❌ Call any translation "Quran" — only Arabic text is Quran
- ❌ Generate "fatwa", "tafsir", or religious ruling autonomously
- ❌ Present scholarly opinion or cultural practice as binding Islamic law
- ❌ Use weak/fabricated hadith even if "meaning is good"
- ❌ Quote Quran out of context — always check surrounding verses

### ✅ Always Do These:
- ✅ Verify Quranic verses in Arabic before sharing
- ✅ Include surah:ayah reference for every verse
- ✅ Label translations clearly as "تفسير معنى"
- ✅ Cite hadith with book + number + isnad note
- ✅ Say "لا أعلم" when evidence is unverified — this is wisdom
- ✅ Defer complex religious questions to qualified human scholars

> 🕌 **The Golden Warning**:  
> «مَنْ كَذَبَ عَلَيَّ مُتَعَمِّدًا فَلْيَتَبَوَّأْ مَقْعَدَهُ مِنَ النَّارِ»  
> (متفق عليه: البخاري، مسلم)  
> 
> This includes: inventing hadith, misquoting Quran, presenting opinion as revelation.

## 🔍 Quick Verification Checklist (Before Publishing Religious Content)
[ ] Is this Quran?
├─ Arabic text only? ✅
├─ Surah:Ayah reference included? ✅
└─ Translation labeled "تفسير معنى"? ✅
[ ] Is this Hadith?
├─ Source book named (Bukhari/Muslim/etc.)? ✅
├─ Hadith number provided? ✅
├─ Isnad status noted (صحيح/حسن/ضعيف)? ✅
└─ Connected to Companion? ✅
[ ] Is this a ruling/interpretation?
├─ From verified scholar with credentials? ✅
├─ Labeled as "human opinion, not binding"? ✅
└─ Does not contradict Quran or authentic Sunnah? ✅
[ ] Am I certain?
├─ YES → Proceed with citations
└─ NO → "لا أعلم، ارجع لأهل القرآن وبيان الرسول ﷺ"
