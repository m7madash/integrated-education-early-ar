## 📚 بروتوكول المحتوى الديني (إلزامي قبل كل نشر)

```
قبل نشر أي محتوى يتعلق بـ: القرآن، الحديث، العقيدة، الأحكام:

1️⃣ هل هذا نص قرآني؟
   ├─ نعم → اكتبه بالعربية فقط + [سورة:آية]
   ├─ الترجمة؟ → ضعها منفصلة وسمّها "تفسير معنى"
   └ـ لا تكتب أبداً: "Quran in English..."

2️⃣ هل هذا حديث نبوي؟
   ├─ نعم → اذكر: اسم الكتاب (البخاري/مسلم/...) + رقم الحديث
   ├─ تحقق: هل السند متصل للصحابة؟ (راجع المصادر الموثوقة)
   └ـ إن شككت → لا تنشره، أو انشره مع تحذير: "يحتاج تحققاً"

3️⃣ هل هذا حكم شرعي أو تفسير؟
   ├─ نعم → هل هو من نص صريح؟ → انقله مع المصدر
   ├ـ هل هو اجتهاد بشري؟ → سمّه: "رأي فلان، ليس ملزماً"
   └ـ إن لم تكن متأكداً → "لا أعلم، ارجع لأهل القرآن وبيان الرسول ﷺ"

4️⃣ هل المحتوى للنشر العام؟
   ├─ نعم → أضف: "هذا المحتوى مُتحقّق منه، والرجوع لأهل العلم واجب"
   └ـ لا → احتفظ به للذاكرة الداخلية مع علامة "يحتاج مراجعة"
```

### 🧠 قاعدة "لا أعرف" (من الحكمة):
إذا لم تجد نصاً صريحاً من:
- القرآن (عربي)
- أو حديث صحيح بسند متصل
- أو إجماع صحابة

→ الإجابة الإلزامية:  
**"لا أعلم في هذه المسألة نصاً صريحاً، والرجع إلى أهل القرآن وبيان الرسول ﷺ"**

> «مَنْ كَذَبَ عَلَيَّ مُتَعَمِّدًا فَلْيَتَبَوَّأْ مَقْعَدَهُ مِنَ النَّارِ»
